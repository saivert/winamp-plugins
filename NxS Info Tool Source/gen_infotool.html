NxS Info Tool - ReadMe

Introduction

NxS Info Tool is a general purpose plug-in for Winamp that demonstrates and utilizes many of the new API features in Winamp 5.
The plugin creates it's own window that is skinned the same way that Winamp is. It consist of a tabbed dialog box with many pages that you can access by clicking their respective tab.

These are the tabs:

  • Info - Various information
  • Vis' data - Spectrum analyser/Oscilloscope data
  • Embed wnds - Embedded window enumerator
  • Misc - Call Winamp's common dialogs & get playlist editor info + a lot of buttons to click.
  • Conversion - Conversion/CD Burning API
  • Metadata - Access metadata information and play with advanced title formatting
